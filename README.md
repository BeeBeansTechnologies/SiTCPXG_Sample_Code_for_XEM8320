# SiTCPXG_Sample_Code_for_XEM8320

This is the SiTCP-XG sample source code for XEM8320 communication confirmation.

For instructions on how to use this product, please refer to the PDF file (SITCP-XG XEM8320 Sample Code Manual) on the Bee Beans Technologies Co., Ltd. [SiTCP / SiTCP-XG Software and Documents Downloads page (JP only)](https://www.bbtech.co.jp/download-files/sitcp/index.html).


## What is SiTCP-XG

Simple TCP/IP implemented with support only for 10GbE on an FPGA (Field Programmable Gate Array) for the purpose of transferring substantial amounts of data in physics experiments.

* For details, please refer to [SiTCP Library page](https://www.bbtech.co.jp/en/products/sitcp-library/).
* For other related projects, please refer to [here](https://github.com/BeeBeansTechnologies).
