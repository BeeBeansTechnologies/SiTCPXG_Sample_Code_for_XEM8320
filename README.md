# SiTCPXG_Sample_Code_for_XEM8320

This is the SiTCPXG sample source code for XEM8320 communication confirmation.

Please check the PDF file (XEM8320_SiTCP_XG_EEPROM.pdf) for how to use it.


## What is SiTCPXG

Simple TCP/IP implemented with support only for 10GbE on an FPGA (Field Programmable Gate Array) for the purpose of transferring large amounts of data in physics experiments.

* For details, please refer to [SiTCP Library page](https://www.bbtech.co.jp/en/products/sitcp-library/).
* For other related projects, please refer to [here](https://github.com/BeeBeansTechnologies).
